# Website Component Catalog

## Navigation Components

### Primary Navigation
- **Header Navigation Bar**: Main site navigation with logo and primary links
- **Hamburger Menu**: Mobile navigation toggle
- **Mega Menu**: Multi-column dropdown for complex site structures
- **Sticky/Fixed Navigation**: Navigation that follows scroll
- **Off-Canvas Menu**: Slide-in navigation drawer
- **Tab Navigation**: Horizontal or vertical tabs for content sections

### Secondary Navigation
- **Breadcrumbs**: Path showing current location in site hierarchy
- **Pagination**: Page numbers for multi-page content
- **Table of Contents**: Jump links to page sections
- **Side Navigation**: Vertical navigation for sub-sections
- **Footer Navigation**: Site map and secondary links
- **Language Selector**: Multi-language site switcher

### Utility Navigation
- **Search Bar**: Site-wide search functionality
- **User Account Menu**: Login/profile dropdown
- **Shopping Cart Icon**: E-commerce cart indicator
- **Notification Bell**: Alert indicator
- **Settings Gear**: Preferences access
- **Help/Support Link**: Access to documentation
